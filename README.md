# myband-front
myband前端部分

## 各文件夹的作用

| 文件夹 | 说明 |
|:---:|:---:|
| / | html文件 |
| /css | css |
| /script | 各种脚本 |
| /images | 图片 |
| /music | 音乐 |

## 文件命名规范


## 特殊文件

|文件名|功能|
|:---:|:---:|
|basic.css|全局css文件，所有页面都引用|
|basic.js|全局js文件，所有页面引用|
|[页面名].css|各页面对应的css文件|
|[页面名].js|各页面对应的js文件|
